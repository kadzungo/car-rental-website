# car-rental-website
Car Rental Management System

A web-based Car Rental Management System that allows administrators to manage vehicles, customers, bookings, and payments through an easy-to-use dashboard.

The system helps rental companies track available vehicles, process reservations, and manage customer information efficiently.

Project Overview

The Car Rental System provides a centralized platform where rental businesses can manage their fleet, customer bookings, and payment records.

It simplifies the rental process by allowing administrators to monitor vehicle availability and manage reservations in real time.

Features
Vehicle Management

Add new vehicles to the fleet

Update vehicle details

Remove vehicles from the system

Track vehicle availability

Customer Management

Register customers

Update customer information

View customer rental history

Booking System

Create new bookings

Assign vehicles to customers

Track booking dates and return dates

Payment Management

Record rental payments

View payment history

Generate billing records

Admin Dashboard

Overview of active rentals

Vehicle availability statistics

Revenue tracking
